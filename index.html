Nagel-Schreckenberg process

  • Let $0 < p < 1$, and $v_{max} \in \mathbb N$
  • Let $x_n$ and $v_n$ be the position and speed of the $n$th vehicle
  • Let $d_n$ be the gap in front of the $n$th vehicle

    • Update rules

  • $v_n = \min\left(v_n + 1, v_{max}\right)$
  • $v_n = \min\left(v_n, d_n\right)$
  • $v_n = v_n - 1$ with probability $p$
  • $x_n = x_n + v_n$
Nagel, Schreckenberg, J Phys (1992) Acceleration of free cars, deceleration due to other cars, randomisation, movement

Nagel-Schreckenberg process

    Realistic parameters

  • time step 1 sec
  • cell size $\sim 7.5 \text{m}$
  • $v_{max} = 5$ corresponds to 135 km/h

  • Very simple model of a driven system
  • Likely in the KPZ universality class

Dynamic lane changing

  • If $\min\left(\color{#1b9e77}{v_A} + 1, d_A^{(f)}, v_{max}\right) > \min\left(\color{#1b9e77}{v_A} + 1, d_A, v_{max}\right)$ then lane change is desirable
  • If $d_A^{(b)} \geq \color{#d95f02}{v_B}$ the lane change is safe
  • If desirable and safe, accept with probabiltiy $p_{change}$

Topological lane changing

  • A is not allowed to change lane, not required to
  • B is allowed to change lane, not required to
  • C is allowed to change lane, required to

Dynamical lane changing allowed if it doesn't contradict topological lane changing

Turning decisions

  • Random with some input probabilities: $P(l \to l')$
  • Decision made when vehicle first enters a link
  • It affects lane changing dynamics
  • Turning probabilities can be constant or time-dependent

Boundary rates

  • Links are given (time-dependent) inflow and outflow rates
  • "Open" boundaries conditions
